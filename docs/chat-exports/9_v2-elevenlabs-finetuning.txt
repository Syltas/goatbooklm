⏺ Docs gelesen. Kritischer Befund zuerst:

  Blocker: Audio-Tags brauchen eleven_v3

  - Aktuelle Pipeline (lib/studio/elevenlabs.ts:14) nutzt eleven_multilingual_v2
  → versteht KEINE Tags. [laughs] würde vorgelesen oder ignoriert.
  - eleven_v3: Tags ✓, 70+ Sprachen inkl. Deutsch, 5.000 Zeichen/Request, läuft
  auch über normalen TTS-Endpoint (nicht nur Text-to-Dialogue).
  - Tag-Arsenal: [laughs], [sighs], [whispers], [exhales], [clears throat],
  [curious], [excited], [sarcastic], [interrupting]/[jumping in] +
  Punctuation-Steuerung (Ellipsen = Pausen, Gedankenstrich =
  Abbruch/Unterbrechung, CAPS = Betonung).
  - Stability bei v3 nur 3 Werte: Creative (expressiv, halluzinationsanfällig) /
  Natural / Robust. Empfehlung: Natural.

  Overlaps — echte Voice-Mischung

  - Per-Turn-TTS + Concat (aktuelle Architektur) → Overlap unmöglich ohne
  ffmpeg-Mixing im Worker. Teuer, will ich nicht.
  - Aber: Text-to-Dialogue-API rendert beide Stimmen in EINEM Stream →
  Unterbrechungen/Überlappungs-Timing macht ElevenLabs selbst. Wurde damals
  verworfen wegen 2.000-Zeichen-GESAMT-Limit (elevenlabs.ts:6-10) — Lösung:
  Skript in ~1.800-Zeichen-Blöcke schneiden, pro Block ein Dialogue-Call, Blöcke
  concatenaten. Segment = Block statt Turn, Resume-Logik im Worker bleibt
  gleich. Bonus: Turn-Übergänge INNERHALB eines Blocks klingen natürlich statt
  gestitcht.

  Deine Liste + was noch fehlt

  Deine Punkte (ähm/mhm, Zuhörer-Zustimmung, Unterbrechungen, unvollständige
  Sätze, Lachen/Seufzen, Overlaps) — alle machbar. Ergänzungen:

  - Host-Personas: Sprecher 1 = neugierig-enthusiastisch, Sprecher 2 =
  analytisch-erdend. Ohne feste Rollen klingt jeder Turn austauschbar.
  NotebookLMs eigentliches Geheimnis.
  - Backchannel als eigene Mikro-Turns: "Mhm", "Ja genau", "Warte —" (1–3
  Wörter), nicht nur in Sätze eingebaut.
  - False Starts + Selbstkorrektur: "Das ist — also eigentlich sind's zwei
  Sachen."
  - Analogien/Alltagsvergleiche für abstrakte Konzepte —
  NotebookLM-Signature-Move.
  - Callbacks: später im Gespräch auf früheren Moment zurückkommen ("wie du
  vorhin gesagt hast…").
  - Zuhörer direkt adressieren: "Und falls ihr jetzt denkt…"
  - Spannungsbogen: Cold Open (habt ihr), Midpoint-"Moment, WAS?"-Moment, Ende =
  Takeaway + offene Frage.
  - Tag-Sparsamkeit: Docs warnen — Tags wirken pro Voice unterschiedlich, Spam
  killt Qualität. ~1 Tag alle paar Turns.

  Risiken (vor Umbau prüfen)

  1. v3 + Deutsch + George/Sarah: Tag-Response ist voice-abhängig,
  Professional-Clones schwächer — ob Premade-Voices auf Deutsch mit Tags gut
  klingen = empirische Frage.
  2. Quota: Konto lief zuletzt auf quota_exceeded (10k-Limit).

  3. previous_text/next_text mit v3: Support unverifiziert — multilingual_v2
